## Charlotte Interactive Land Use Webmap

Data Source: NLCD Landcover Data (1985 - 2024)


Data Processing:

- Download the data.
- Clip the data using the Mecklenburg County census tract level boundary utilizing ArcPy.
- Calculate each class total pixel using customize python programing in ArcPy.
- Reclassify the classes into broader categories: Impervious, Forest, Waterbody, and Low Vegetation.
